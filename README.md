# Divine Portfolio

A beautifully designed, professional fashion portfolio website built to showcase designer collections, sell curated pieces, and attract clients with premium visual storytelling.

## Overview

`Divine Portfolio` is a modern web experience tailored for fashion designers, stylists, and creative directors. It blends editorial-grade presentation with refined e-commerce flow, allowing brands to display lookbooks, highlight signature collections, and convert visitors into clients with confidence.

## Key Features

- Elegant home and portfolio sections with immersive imagery
- Professional product displays for fashion collections
- Smooth scroll reveal animations and refined hover interactions
- Responsive layout built for both desktop and mobile
- Integrated cart and checkout-ready structure for selling fashion pieces
- Designer-oriented navigation for brand storytelling and client discovery
- Admin login panel for secure product management and curated collection updates

## Admin Section

The admin area enables designers or brand managers to log in securely, add new products, and manage the portfolio inventory. New items entered through the dashboard appear within the collection and portfolio pages, giving clients an updated and professional shopping experience.

## Why it stands out

This website is crafted to feel like a polished fashion brand experience. The portfolio cards appear with graceful motion and staggered reveals, while the content remains clean and sophisticated so that each design, garment, or collection shines.

## Installation

Follow these steps to clone the project from GitHub and set up the React application locally:

1. Open a terminal and clone the repository:
   ```bash
   git clone https://github.com/aviavinas01/phii.git
   ```
2. Change into the project directory:
   ```bash
   cd phii
   ```
3. Install the client dependencies:
   ```bash
   cd client
   npm install
   ```
4. If you are also using the backend, open a second terminal and install server dependencies:
   ```bash
   cd phii/server
   npm install
   ```
5. Start the React development server:
   ```bash
   cd phii/client
   npm run dev
   ```
6. Open your browser and visit `http://localhost:5173` to view the site.

If you have npm installed but prefer Yarn or pnpm, you can replace `npm install` with `yarn install` or `pnpm install`.
